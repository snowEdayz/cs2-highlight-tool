## 中文

### 新增

- 更新软件后首次进入主界面时，会弹窗展示当前版本的更新日志，关闭后同版本不再重复弹出
- 弹窗内容支持中英双语，按当前界面语言自动切换

### 优化

- 当前版本若没有匹配的更新日志文件，弹窗会静默跳过，不再打断用户

## English

### Added

- After updating the app, a changelog modal is shown on the first launch into the main app; it will not appear again for the same version once dismissed
- The modal supports both Chinese and English content and follows the current UI locale

### Improved

- If the current version has no matching changelog file, the modal is silently skipped instead of interrupting the user
